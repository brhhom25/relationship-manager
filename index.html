WhatsApp Business API

Use Meta's Cloud API (free tier) to send & receive messages directly.

Setup: 1) Create Meta App → Add WhatsApp product → Get Phone Number ID & Token. 2) Set webhook URL to your hosted page + /wa-webhook. 3) Paste token above & click Save.

Telegram Bot

Create a bot via @BotFather and connect it to read & send messages.

Setup: 1) Message @BotFather → /newbot → copy token. 2) Enter token + webhook URL above. 3) Click Activate — the app will register the webhook with Telegram automatically.
Not connected

Google Contacts Sync

Two-way sync: import from Google Contacts and push updates back.

Create credentials at console.cloud.google.com → APIs & Services → Credentials → OAuth 2.0 Client ID (Web). Enable the People API. Set authorized origin to this page's URL.

Voice Commands

Uses the browser's built-in Web Speech API. Works best in Chrome/Edge. Activate with the microphone button in the top bar, or enable always-on below.

Available Commands

"go to contacts"→ Switch to Contacts tab
"open dashboard"→ Switch to Dashboard tab
"start recording"→ Start live call recording
"stop recording"→ Stop live recording
"add contact [name]"→ Open Add Contact & pre-fill name
"find [name]"→ Search contacts
"follow ups"→ Switch to Follow-Ups tab
"generate insights"→ Run AI follow-up generation
"mark [word]"→ Add timestamped call marker
